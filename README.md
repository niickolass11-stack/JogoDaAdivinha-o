# JogoDaAdivinhacao

# e recebe tambem a mensagem dizendo quantas tentativas ainda lhe restam
# a fim do jogo são adidiconados alguns pontos para o jogador caso ele acerte o numero
# se o jogador quiser ele pode acessar os pontos e ver quantos pontos consegui fazer
